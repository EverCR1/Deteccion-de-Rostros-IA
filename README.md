**Pasos para Instalar**


Crear entorno
```
python -m venv venv
```

Activar entorno
```
venv\scripts\activate
```

Instalar dblib necesario para face recognition
```
pip install dlib-19.19.0-cp38-cp38-win_amd64.whl
```

Instalar librerías
```
python -m pip install -r requirements.txt
```

Comprobar librerías instaladas
```
pip freeze | pip list
```
